# Proyecto-final-DevF

**Descripción**

Este proyecto final es haciendo una pagina web a la banda llamada Bad Omens. De manera sencilla se integra una breve historia de la banda, los albumes que han sacado, los miembros de la banda y manera de contactarlos a traves de sus redes y su página oficial. Tambien se agregaron enlaces para acceder a los albumes y un pequeño formulario simulando que quisieras obtener mas información de la banda.

**Caraterísticas**

Para este proyecto se utilizaron distintas etiquetas, entre ellas la parte de section en vez de div o article en vez de un div, ya que no solo es una division, estamos diciendo que es una seccion completa. Albumes y miembros son una seccion completa y article tiene el mismo formato de una imagen y una leyenda, no es solo una division de 3 albumes y 3 miembros.

Para el estilo fue un estilo sencillo donde se investigo de manera independiente, en vez de que las imagenes estan grandes se les dio un tamaño y un orden en una linea horizontal, eso para las secciones de album y miembros. 

**Como desplegarlo**

Primero se crea un repositorio en GitHub, una vez creado puedes bajar el respositorio con comandos en GitBash con la url de tu repositorio. En este caso se creo una rama llamada develop para trabajar en ello y despues pasarla al main y de esa manera la version buena esta en el main y en develop puedo realizar todos los cambios antes de mostrarlo a la pagina.

Una vez que hemos pasado los archivos correspondientes al main iremos a la seccion de settings o configuracion, despues iremos a GitHub Pages y elegimos una rama, la rama sera main y con eso tendremos un enlace que nos lleva a la pagina publicada en GitHub Pages con los archivos que tenemos en la rama main.
